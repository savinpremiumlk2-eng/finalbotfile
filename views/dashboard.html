User Manual / Guide

Getting Started

  1. Go to Deploy Bot from the sidebar
  2. Choose a connection method: Session ID, Pair Code, or QR Code
  3. Once connected, the bot will appear in Bot Sessions
  4. Customize behavior in Settings

Key Commands

.menu - Main menu (reply with number)
.settings - View/change bot settings
.mode - Switch private/public mode
.antiviewonce - Anti view-once toggle
.sticker - Create stickers
.song - Download songs
.ai - AI chat
.film2 - Search movies

Connection Methods

  • Session ID - Paste a KnightBot session string
  • Pair Code - Enter your phone number, get a code, enter it in WhatsApp > Linked Devices > Link with phone number
  • QR Code - Scan from WhatsApp > Linked Devices > Link a Device

ආරම්භ කිරීම

  1. Sidebar එකෙන් Deploy Bot වෙත යන්න
  2. සම්බන්ධ කිරීමේ ක්‍රමයක් තෝරන්න: Session ID, Pair Code, හෝ QR Code
  3. සම්බන්ධ වූ පසු, බොට් එක Bot Sessions හි පෙනෙනු ඇත
  4. Settings හි බොට් එකේ හැසිරීම් අභිරුචිකරණය කරන්න

ප්‍රධාන විධාන

.menu - ප්‍රධාන මෙනුව (අංකයෙන් reply කරන්න)
.settings - බොට් සැකසුම් බලන්න/වෙනස් කරන්න
.mode - Private/Public මාදිලිය මාරු කරන්න
.antiviewonce - View-once ආරක්ෂාව on/off
.sticker - Sticker සාදන්න
.song - ගීත බාගන්න
.ai - AI Chat
.film2 - චිත්‍රපට සොයන්න

සම්බන්ධ කිරීමේ ක්‍රම

  • Session ID - KnightBot session string එකක් paste කරන්න
  • Pair Code - දුරකථන අංකය ඇතුළත් කරන්න, code එකක් ලැබේ, WhatsApp > Linked Devices > Link with phone number වල ඇතුළත් කරන්න
  • QR Code - WhatsApp > Linked Devices > Link a Device වලින් scan කරන්න

ඉඟි

  • .menu ටයිප් කළ පසු, submenu එකකට යාමට අංකයක් reply කරන්න
  • Submenu එකේ command එකක් ක්‍රියාත්මක කිරීමට එහි අංකය reply කරන්න
  • ආපසු main menu එකට යාමට 0 reply කරන්න
  • සියලු settings එකවර බැලීමට .settings භාවිතා කරන්න
